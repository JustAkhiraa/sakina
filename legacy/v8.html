À propos du Qadâ'
وَأَقِيمُوا الصَّلَاةَ وَآتُوا الزَّكَاةَ

Le Qadâ' consiste à rattraper les prières obligatoires manquées. Selon les savants, il est obligatoire de les rattraper dès que possible.

Comment utiliser : Entrez le nombre de jours ou d'années pendant lesquels vous avez manqué vos prières. Le nombre de prières manquées par prière sera calculé automatiquement.

Pour chaque prière rattrapée, appuyez sur −. Votre progression est sauvegardée automatiquement.
